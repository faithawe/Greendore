Tab transitions & navigation behaviour

How the app moves between tabs

Tab switch animationScreen content crossfades — opacity 0→1, 200ms ease-out. No slide. Tabs are not ordered sequentially so a slide direction is meaningless.
Icon tap feedbackIcon scale 1→1.12→1.0 over 160ms. Active bg fades in 120ms.
Scroll-to-top on re-tapTapping the active tab a second time scrolls the content back to top. Standard iOS pattern, expected by users.
Nav visibilityNav is always visible — never hides on scroll. Content scrolls behind it. Scroll body has padding-bottom: 100px so last item is never hidden.
Header color per tabGarden: #2D4A30 Canopy · Grow: #2D4A30 Canopy · Harvest: #3A2818 dark terra · Community: #1C2838 deep navy
Nav bg adaptsWhen a tab has a dark header that fills the screen, nav switches to dark treatment. Linen tabs use light frosted nav.
Profile/Settings accessAvatar in Garden header only. Tapping opens a full-screen slide-up sheet — does not change active tab or tab state.
Back navigationSheets (sensor detail, task detail, harvest log, recipe) use drag-to-dismiss + close button. No back arrow in nav bar — keeps header clean.

Tab icon reference — SVG paths

Exact icon design for each tab, inactive and active stroke color

Garden iconHouse silhouette. Path: M10 2L4 7v9a1 1 0 001 1h4v-5h2v5h4a1 1 0 001-1V7L10 2z. No fill, outline only. Represents "home base" for the plant.
Grow iconLeaf + stem. Teardrop path from bottom point up + a short stem with horizontal end-cap. Represents growth and the plant lifecycle.
Harvest iconMountain/yield graph shape + circle above. Rising peaks reference harvest data over time. Circle = the fruit/yield.
Community iconTwo overlapping people silhouettes — larger in front, smaller behind. Foreground person fully opaque, background person at 80% opacity.
Stroke width1.4px for all icons. Consistent visual weight across all 4. No filled icons — always outline.
Icon design ruleIcons are legible at 20×20px with no label. The label confirms — it doesn't explain. If the icon needs the label to be understood, redesign the icon.
